# CIFAR-10 Image Classification using ResNet18

## Overview
This project implements a deep learning image classifier using PyTorch and transfer learning with a pretrained ResNet18 model on the CIFAR-10 dataset.

## Features
- Transfer Learning (Pretrained ResNet18)
- Data Augmentation
- Training Curve Visualization
- Evaluation Metrics
- Confusion Matrix
- Model Inference

## Installation

pip install -r requirements.txt

## Training

python train.py

## Evaluation

python evaluate.py

## Inference

python inference.py

## Results

Test Accuracy: 80.41%

Metrics:
- Precision
- Recall
- F1-Score

- outputs/training_curve.png
